## 安装

> npm install morning-ui --save

## 快速开始

    // 引入Vue.js
    import Vue from 'vue';

    // 引入Morning UI
    import morning from 'morning-ui';
    import 'morning-ui/dist/morning-ui.css';

    // 初始化Morning UI
    Vue.use(morning);

接下来即可像使用原生HTML一样使用Morning UI提供的组件，更多信息查看：[使用文档](https://morning-ui.com/guide/usage.html)。
